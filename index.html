WELCOME TO D -TECH

     D - TECH ผู้ผลิตและจัดจำหน่ายสินค้าด้านระบบอัตโนมัติทางอุตสาหกรรม Industrial Communication Gateway และ Industrial IoT เรามีสินค้าและบริการสำหรับงานอุตสาหกรรม เรามุ่งมั่นให้บริการและส่งมอบสิ่งที่ดีที่สุดให้กับลูกค้า ในโรงงานอุตสาหกรรมต่าง ๆ อาทิเช่น อุตสาหกรรมไฟฟ้า การควบคุมอัตโนมัติการควบคุมเครื่องจักร สนับสนุนการสั่งงานระยะไกล อุตสาหกรรมสำหรับเมืองอัจฉริยะ เราพร้อมให้คำปรึกษาทางด้านการออกแบบระบบเพื่อตอบสนองความต้องการของลูกค้า เราได้มุ่งมั่นที่จะสร้างความพึงพอใจให้แก่ลูกค้าในการเสนอสินค้าและบริการที่มีคุณภาพตลอดระยะเวลาการรับประกันสินค้า ทีมวิศวกรของเราจะค่อยให้คำปรึกษา แนะนำสินค้า และ Solution ที่หลากหลาย ให้ตรงกับความต้องการของลูกค้าเพื่อความพึงพอใจสูงสุดของลูกค้า

ทำไมต้องเลือกซื้อสิ้นค้ากับ D - TECH

เราเป็นผู้ผลิต ออกแบบ และติดตั้งระบบซึ่งมีความพร้อมในการสนับสนุนความต้องการทุกรูปแบบของลูกค้า

ทีมวิศวกรผู้เชียวชาญช่วยให้คำแนะนำที่มีคุณภาพเกี่ยวกับสินค้าและบริการต่างๆ ให้คำแนะนำที่เป็นประโยชน์และเหมาะสมกับความต้องการของตัวคุณ

มีทีม Service บริการที่ครอบคลุมยอดเยี่ยมจากทีมงานก่อนและหลังการขาย

เราทำการทดสอบคุณภาพสินค้าอย่างละเอียดทุกชิ้นก่อนจัดส่งให้กับลูกค้า


    COMPACT RTU เป็นอุปกรณ์ที่ออกแบบมาให้สามารถเชื่อมต่อเข้ากับระบบบริหารจัดการหลายรูปแบบซึ่งในเทคโนโลยีปัจจุบันเริ่มมีการนำอุปกรณ์ต่างๆ ออกแบบให้เป็นอุปกรณ์ IOT และควบคุมสั่งการผ่าน Software เพื่อให้สามารถบริหารจัดการแบบรวมศูนย์ สามารถติดตามและสั่งงานผ่าน Command Center ประกอบกับอุปกรณ์ในภาคอุตสาหกรรมอุปกรณ์ไฟฟ้าหลายประเภทมีต้นทุนในการเปลี่ยนอุปกรณ์ทั้งหมดเป็นแบบ IOT จึงต้องทำการใส่อุปกรณ์ควบคุมและเชื่อมต่อเข้ากับวงจรไฟฟ้าของอุปกรณ์ เพื่อสั่งการจากระยะไกลพร้อม แจ้งเตือนสถานการณ์ทำงานที่ผิดปกติ ดังนั้น COMPACT RTU จึงถูกนำเข้ามาสร้างระบบควบคุม และสั่งการได้อย่างมีประสิทธิภาพ


    ตัวอย่างการนำ COMPACT RTU สามารถใช้ในการควบคุมสั่งการ อุปกรณ์ไฟฟ้าทั้งภายใน ภายนอกอาคาร ออกแบบให้เชื่อมต่อกับ วงจรที่ใช้กระแสสูงมี Input สามารถป้อนสัญญาณจากอุปกรณ์ กลับมาตรวจสอบสถานการณ์ทำงานของอุปกรณ์ได้ สามารถนำไป ใช้ควบคุมมอเตอร์ ปั๊มน้ำ พัดลมดูดอากาศ หรือระบบแสงสว่าง เชื่อมต่อ Network Protocol TCP/IP ผ่านโครงข่ายสื่อสารภาย ในองค์กร หรือ 3G 4G


    ตัวอย่างการนำ COMPACT RTU ไปใช้ในการสั่งการไฟถนน ซึ่งไฟถนนในพื้นที่ส่วนบุคคล หรือพื้นที่สาธารณะ จะมีจำนวน โคมไฟจำนวนมาก และต้องคอยบริหารจัดการในเรื่องการซ่อม บำรุงหลอดติดดับ และต้องมีการปิด-เปิดไฟถนนตามเวลาในช่วง ค่ำการ ออกแบบให้ระบบมีการสั่งการ และควบคุมหลอดติดดับ ในระยะไกลจึงเป็นประโยชน์ในการบริหารจัดการพื้นที่ สามารถ ประหยัดไฟ จากการเลือกปิดไฟในบริเวณที่ไม่ได้ใช้งาน และส่งค่า พลังงานกลับไปยัง Command Center เพื่อทำการติดตามและ บริหารจัดการ
